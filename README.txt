Firstly, the "Servidor" must be executed so the server runs. Then, "Registro" must be
started as many times as users are wanted in the system. Each client can add other
user by knowing their name, and then obtain the IP adress, that is the field
necessary to initiate a conversation with another user. Once this is done, information
is introduced in the window of "InicioCharla" and the other user must confirm who he
wants to talk to. Then, the conversation is initiated between both users.
